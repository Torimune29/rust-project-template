## check

### description

check files using pre-commit

### example

```bash
check README.md LICENSE
```

## check-all

### description

check all files using pre-commit

### example

```bash

```

## reload-env

### description

Reload flake.
        If reload not flake.nix but .nix, nix-direnv does not reload nix env.
        So run this to reload nix env force.

### example

```bash
reload-env
```
